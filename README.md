# LiftOps Dashboard — Demo

A self-contained HTML dashboard for tracking lift breakdowns, mantraps, and 
vendor performance across a property portfolio. Fictitious data.

## Features
- Portfolio KPIs with month-over-month delta
- Accordion drill-down by site with 12-month history
- Auto-generated AI insights (rules-based)
- Excel import (drop in any compatible .xlsx)
- BU colour-coding · Vendor benchmark · Repeat-offender detection

## Tech
Vanilla JS · Chart.js · SheetJS · LocalStorage · zero build step

## Live demo
Open `index.html` in any modern browser.
